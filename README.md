# 🚗 ParkEase - Parking Management System
ParkEase is a web-based application that allows users to pre-book parking slots online, reducing congestion and saving time. The system provides separate modules for Users and Admins, each with secure login and registration functionality.

---
## 📌 Features
### 👤 User Module
- User registration & login  
- View available parking slots  
- Book parking slots in advance  
- View booking details  
- Responsive and user-friendly interface  

### 🛠️ Admin Module  
- Admin registration & login   
- Manage parking slots (add/remove/update)  
- Monitor booked and available slots  
- Control overall parking availability  

---
## 🧱 Tech Stack
### Frontend  
- HTML5  
- CSS3  
- JavaScript  

### Backend 
- Node.js, Relational DB
- MySQL (for storing users, admins & bookings)

---
## 🚀 Future Enhancements
- Real-time slot availability  
- Payment gateway integration  
- Mobile app version  
- Email/SMS booking confirmation  
- Admin analytics dashboard
